  • 🔗
    PR blame — Traces failure to the exact production PR that introduced it
  • 🛡️
    Deliberate fixes — Documents what was NOT done. No force:true, no hardcoded waits

Evolution

Refined through daily use

Every mistake and edge case made the skills smarter. Here's how they evolved.

01

Research before code

Early iterations guessed at selectors. Now: search prod repos for data-test, check local sandbox first — local tests reveal runtime patterns source code doesn't show.

02

Never skip steps

Shortcutting to "just fix it" caused repeat iterations. Now the full pipeline is enforced: fetch → analyze → research → fix → verify. Step 3 often reveals patterns that change the fix.

03

Stress testing is non-negotiable

A fix that passes once means nothing. Every PR gets 20–60x loops in CI. The skill polls TeamCity until all iterations pass, then removes the loop automatically.

04

Trace to the causing PR

PR #2166 taught the skill to build full timelines — who changed what, when, and why. Not "something changed." Makes root cause undeniable.

05

"What we did NOT do"

PRs now document deliberate non-choices. No force:true, no cy.wait(ms). Future maintainers understand the reasoning, not just the code.

06

Local sandbox first

Creating tests now checks existing sandbox specs before source repos. Local tests reveal iframes, specific URLs, auth patterns, and runtime quirks.

07

Comprehensive assertion checklist

Not just "element visible." Text content, tooltips, modals, navigation state, tracking params. The checklist grew from real missed bugs in production.

08

CodeRabbit integration

Skills poll for review comments, triage by severity, apply direct fixes with commit hashes, and reply with reasoning. The full PR lifecycle is automated.

09

Pattern library compounds

Each investigation adds to cypress-core/patterns/. Started with timing and selectors — now includes auth context, external dependencies, deterministic selection, text matching, migration patterns.

What's next

The skills keep learning

120s initial wait, then 270s intervals. Distinguishes CI progress % from actual run status — no false completions.

SHIPPED CodeRabbit triage

Classifies comments P0–P3. Direct fixes for simple issues, agent dispatch for complex multi-file changes. Replies with commit hashes.

SHIPPED Pattern analysis

analyze-patterns.py and find-causing-pr.py — automated bad pattern detection and production PR tracing.

NEXT Cypress Cloud auth

MCP server auth setup will unlock flaky test history lookup, run analytics, and cross-correlation with TeamCity failures.

NEXT Selector database

Pre-indexed data-test attributes across all prod repos. Cuts research time from minutes to instant lookup.

LATER Confidence auto-proceed

For simple cases (text changes, selector updates), auto-proceed without step-by-step approval. Trust calibrated by pattern match confidence.
