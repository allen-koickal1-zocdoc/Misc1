# Misc Claude Tools

A collection of macOS utilities for launching Claude Code.

## Claude Sandbox (Menu Bar App)

A menu bar app for quickly launching Claude Code in any directory.

**Features:**
- Menu bar icon for quick access
- Global keyboard shortcut (⌘⇧C)
- Recent directories list
- Drag & drop folder support
- iTerm with Terminal.app fallback
- Notifications on launch

**Installation:**
```bash
cd claude-sandbox
./install.sh
```

**Dependencies:** rumps, pyobjc

## Claude Buddy (Desktop Companion)

A walking desktop companion that opens Claude Code on click.

**Features:**
- Animated character walks across the bottom of your screen
- Left-click to open Claude in Terminal
- Right-click to quit

**Installation:**
```bash
cd claude-buddy
./install.sh
```

**Dependencies:** PyQt6
